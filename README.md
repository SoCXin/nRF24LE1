### [nRF24LE1简介](https://github.com/SoCXin/nRF24LE1/wiki)

[nRF24LE1](https://github.com/SoCXin/nRF24LE1) 是 [Nordic](https://www.nordicsemi.com/) 集成增强型8051单片机和2.4Ghz低功耗无线收发内核nRF24L01P，是市面上少有的单芯片2.4G射频SoC
